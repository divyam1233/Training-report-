# 🧵 Cat 6 Cable

## 📘 Introduction
**Category 6 (Cat 6) cable** is a standardized twisted pair cable used for Ethernet and other network physical layers.  
It supports **high‑speed data transmission** and is backward compatible with Cat 5/5e standards.


---

# 🔹 1. Features of Cat 6 Cable
- Supports speeds up to **10 Gbps** (over shorter distances ~55m).  
- Bandwidth capacity: **250 MHz**.  
- Uses **RJ45 connectors**.  
- Consists of **4 twisted pairs** of copper wires.  
- Provides reduced **crosstalk** compared to Cat 5e.  

---

# 🔹 2. Cat 6 vs Cat 5e

| Feature        | Cat 5e           | Cat 6            |
|----------------|------------------|------------------|
| Speed          | Up to 1 Gbps     | Up to 10 Gbps    |
| Bandwidth      | 100 MHz          | 250 MHz          |
| Crosstalk      | Moderate         | Reduced (better shielding) |
| Use Case       | Home networks    | Enterprise, high‑speed LAN |

# 🔧 Crimping (RJ45 Cable Preparation)

## 📘 Introduction
**Crimping** is the process of attaching an RJ45 connector to the end of a network cable (Cat5e, Cat6).  
It ensures proper electrical contact between the cable wires and the connector pins, enabling reliable data transmission.
