# 🌐 Structure of an HTML Document

## 📘 Introduction
An **HTML document** follows a standard structure that defines how web browsers interpret and display content.  
It consists of a **DOCTYPE declaration**, `<html>`, `<head>`, and `<body>` sections.

# 🔹 1. Basic Structure

👉 Example:
```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
    <p>This is a paragraph.</p>
</body>
</html>
