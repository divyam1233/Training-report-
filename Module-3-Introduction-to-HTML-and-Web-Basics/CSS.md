# 🎨 CSS (Cascading Style Sheets) Detailed

## 📘 Introduction
**CSS (Cascading Style Sheets)** is used to style and format HTML documents.  
It controls layout, colors, fonts, spacing, and responsiveness of web pages.

# 🔹 1. Features of CSS
- Separates **content (HTML)** from **presentation (CSS)**.  
- Provides **consistency** across multiple pages.  
- Supports **responsive design** for different devices.  
- Enables **animations and transitions**.  

---

# 🔹 2. Types of CSS

| Type | Description | Example |
|------|-------------|---------|
| Inline CSS | Applied directly inside HTML element | `<p style="color:red;">Text</p>` |
| Internal CSS | Defined inside `<style>` tag in `<head>` | `<style> p { color: blue; } </style>` |
| External CSS | Stored in `.css` file and linked | `<link rel="stylesheet" href="style.css">` |
