# 🌐 HTML Basics

## 📘 Introduction
**HTML (HyperText Markup Language)** is the standard language used to create and structure web pages.  
It defines the content of a webpage using **tags** and **elements**.

# 🔹 1. Features of HTML
- Defines the **structure** of a webpage.  
- Uses **tags** enclosed in `< >`.  
- Works with **CSS** (for styling) and **JavaScript** (for interactivity).  
- Platform‑independent and supported by all browsers.  

---

# 🔹 2. Basic HTML Structure

👉 Example:
```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
    <p>This is a paragraph.</p>
</body>
</html>
